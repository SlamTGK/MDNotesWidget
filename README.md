**MD Notes Widget** — это Android-приложение, которое переносит ваш "Второй Мозг" прямо на рабочий стол. Оно случайным образом выбирает и красиво отображает Markdown (`.md`) файлы из любой папки на вашем устройстве. Создано для идеальной интеграции с локальными базами Obsidian и другими Markdown-редакторами.

## ✨ Ключевые возможности

* 🎲 **Умный рандом**: Периодически выводит случайную заметку на экран. Встроенная защита от повторов гарантирует, что вы всегда будете видеть свежий контент.
* 🎨 **Адаптивный и красивый UI**: На выбор доступно три современные темы (Фирменная фиолетовая, Тёмная, Полупрозрачная). Виджет динамически адаптируется под любой размер и элегантно обрезает длинный текст в самом низу.
* ⚡ **Глубокая интеграция с Obsidian**: При клике на виджет приложение напрямую откроет нужную заметку прямо внутри мобильного приложения Obsidian.
* 🏷️ **Фильтрация по тегам**: Хотите видеть только цитаты или карточки для запоминания? Укажите тег (например, `#quote`), и виджет будет генерировать заметки только с этим словом.
* 🚫 **Черный список папок**: Легко исключайте системные или нерелевантные директории (например, `Templates`, `Archive`), чтобы они не попадали в выдачу.
* 🔋 **Нулевой расход батареи**: Построено на современном API `WorkManager`. Приложение полностью "умирает" и засыпает между обновлениями, потребляя ровно 0% аккумулятора и процессора.
* 🔒 **Максимальная приватность**: Приложение полностью оффлайновое. Для работы не требуется доступ в интернет. Использует безопасный SAF (Storage Access Framework) Android'a для доступа строго к одной выбранной вами папке.

## 🛠️ Требования
* **Android 12.0+ (API 31)** и выше (вплоть до Android 16+).
* Идеально адаптировано для работы со строгими оболочками вроде MIUI / HyperOS (отсутствуют краши RemoteViews из-за ограничений в архитектуре).

## 📥 Как установить

1. Перейдите в раздел [Releases](https://github.com/SlamTGK/MDNotesWidget/releases).
2. Скачайте свежий `app-release.apk`.
3. Скачайте и установите APK на ваше Android-устройство.

*(Примечание для пользователей Xiaomi/HyperOS: после установки рекомендуем зайти в настройки приложения и разрешить "Автозапуск" для бесперебойного обновления).*

## ⚙️ Настройка

1. Добавьте виджет **MD Notes Widget** на свободное место на рабочем столе.
2. Нажмите на виджет, чтобы зайти в Настройки.
3. Нажмите **Выбрать папку** и укажите главную директорию с вашим хранилищем (например, Obsidian Vault).
4. Настройте шрифты, темы, интервал или черный список по желанию.
5. Нажмите кнопку **Обновить все виджеты**.

**MD Notes Widget** — это Android-приложение, которое переносит ваш "Второй Мозг" прямо на рабочий стол. Оно случайным образом выбирает и красиво отображает Markdown (`.md`) файлы из любой папки на вашем устройстве. Создано для идеальной интеграции с локальными базами Obsidian и другими Markdown-редакторами.

## Что нового в v2.0

- **Полноэкранный просмотр заметок** со свайпом между случайными заметками, рендерингом Markdown и превью изображений
- **Закрепление заметки** на виджете — кнопка-pin рядом с названием
- **Виджет-список** — новый тип виджета с пролистыванием заметок (StackView)
- **Material You / Monet** — динамические цвета системы на Android 12+
- **Кастомные цвета виджета** — выбор фона, текста и заголовка из палитры
- **Тихие часы** — настраиваемый период, когда виджет не обновляется
- **Несколько тегов + логика И/ИЛИ** — продвинутая фильтрация заметок
- **История просмотренных заметок** — последние 50 заметок с возможностью вернуться
- **Локализация** — полная поддержка русского и английского языков
- **Встроенный просмотрщик** — новый вариант открытия заметок (наряду с Obsidian и системным выбором)
- Множество исправлений и оптимизаций производительности

## Ключевые возможности

### Виджеты
- **Случайная заметка** — классический виджет, периодически показывающий новую заметку
- **Виджет-список** — карточки заметок с пролистыванием свайпом прямо на рабочем столе
- **Закрепление** — нажмите на иконку pin, чтобы заметка осталась на виджете
- **Темы** — Фиолетовая, Тёмная, Прозрачная или свои цвета
- **Размер шрифта** — Мелкий / Средний / Крупный
- **Автоадаптация** — виджет автоматически подстраивает количество строк под свой размер

### Просмотр заметок
- **Полноэкранный просмотрщик** с рендерингом Markdown (заголовки, списки, чекбоксы)
- **Свайп** между случайными заметками
- **Превью изображений** из заметок
- **История** — лента последних 50 просмотренных заметок
- **Открытие** в Obsidian или любом другом приложении

### Настройки
- **SAF папка** — безопасный доступ к конкретной папке через Storage Access Framework
- **Интервал обновления** — от 30 минут до 24 часов, или выключить
- **Фильтрация по тегам** — несколько тегов через `;`, логика И/ИЛИ
- **Чёрный список папок** — исключение директорий (Templates, Archive и т.д.)
- **Тихие часы** — виджет не обновляется ночью (настраиваемый период)
- **Material You** — динамические цвета на Android 12+ (Monet)

### Технические достоинства
- **Kotlin + Coroutines** — асинхронная обработка без блокировки UI
- **Material 3** — современный дизайн с Dynamic Colors
- **Энергоэффективность** — AlarmManager с JSON-кэшированием
- **Полная приватность** — без интернета, без трекинга, только локальные файлы
- **R8 минификация** — оптимизированный размер APK

## Требования

- **Android 12.0+** (API 31) и выше
- Оптимизировано для MIUI / HyperOS

## Установка

1. Перейдите в раздел [Releases](https://github.com/SlamTGK/MDNotesWidget/releases)
2. Скачайте `app-release.apk`
3. Установите APK на устройство

> **Xiaomi / HyperOS**: для стабильной работы включите "Автозапуск" и установите контроль активности в "Нет ограничений".

## Сборка из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/SlamTGK/MDNotesWidget.git
cd MDNotesWidget

# Собрать debug APK (требуется Android SDK и JDK 17)
./gradlew assembleDebug

# Или через GitHub Actions — APK собирается автоматически
```

## Структура проекта

```
app/src/main/java/com/mdnotes/widget/
├── MDNotesApp.kt              # Application — Material You Dynamic Colors
├── MainActivity.kt            # Экран настроек
├── NoteViewerActivity.kt      # Полноэкранный просмотр заметок
├── HistoryActivity.kt         # История просмотренных заметок
├── NoteWidgetProvider.kt      # Провайдер основного виджета
├── NoteListWidgetProvider.kt  # Провайдер виджета-списка
├── NoteListWidgetService.kt   # Сервис данных для StackView
├── PreferencesManager.kt      # Хранение настроек и состояния
├── MarkdownFileScanner.kt     # Сканирование и парсинг .md файлов
└── FileOpener.kt              # Открытие файлов (Obsidian/Система/Viewer)
```

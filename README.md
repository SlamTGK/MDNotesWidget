**MD Notes Widget** — Android-приложение для тех, кто хранит заметки в Markdown. Случайным образом показывает `.md` файлы из выбранной папки прямо на рабочем столе. Отлично работает с **Obsidian**, Logseq и любыми другими локальными Markdown-хранилищами.

## Виджеты

| Виджет | Описание |
|--------|----------|
| **Случайная заметка** | Показывает одну заметку. Адаптирует количество строк под размер виджета |
| **Расширенный виджет** | Показывает заметку целиком с максимальным текстом |

Общие возможности виджетов:
- 📌 **Закрепить** — фиксирует текущую заметку
- ➕ **Создать** — быстрое создание `.md` файла прямо с виджета
- 🔄 **Обновить** — показать другую случайную заметку
- 🏷️ **Статус тегов** — в строке с датой и папкой: `24 апр 2026, Психология, #избранное`
- 🎨 **Темы** — Фиолетовая, Тёмная, Прозрачная, Своя
- 🔡 **Размер шрифта** — Мелкий / Средний / Крупный

## Просмотрщик заметок

- **Markdown-рендеринг** — заголовки, **жирный**, _курсив_, ~~зачёркнутый~~, ☑ чекбоксы, списки
- **Редактор** — кнопка ✏️ переключает в режим редактирования с monospace-шрифтом и автосохранением
- **Изображения** — горизонтальная прокрутка с превью картинок из заметки
- **Свайп** — листайте между случайными заметками
- **Поиск** — живой поиск по имени и содержимому заметок
- **Избранное** — добавить одним нажатием, свайп для удаления
- **История** — последние 50 открытых заметок со свайп-действиями
- **«В этот день»** — заметки из этого дня в прошлые годы
- **Меню ⋮** — поделиться, открыть в Obsidian, открыть внешним приложением, удалить

## Фильтрация и настройки

- **Теги** — введите теги через запятую (`избранное, daily`), виджет покажет только заметки с этими тегами
- **Поиск тегов** — находит `#теги` и в YAML frontmatter, и прямо в тексте заметки
- **Авто-подсказки** — ChipGroup показывает теги из ваших реальных заметок
- **Чёрный список папок** — исключить папки из выборки (`.trash`, `templates` и т.д.)
- **Антиповтор** — последние 20 показанных заметок не повторяются в случайной выборке
- **Интервал обновления** — выкл / 15 мин / 30 мин / 1–24 ч
- **Тихие часы** — виджет не обновляется ночью
- **Действие при клике** — встроенный просмотрщик / Obsidian / системный выбор

## Установка

1. Скачайте APK из [Releases](../../releases/latest)
2. Разрешите установку из неизвестных источников
3. Установите APK
4. Долгим нажатием на рабочем столе добавьте виджет **MD Notes Widget**
5. В настройках виджета выберите папку с `.md` файлами

> **Xiaomi / HyperOS:** Для стабильной работы включите «Автозапуск» и снимите ограничения батареи для этого приложения.

## Требования

- Android 12+ (API 31)
- Разрешение на доступ к папке (через системный диалог, без root)

## Технологии

- **Kotlin** + Android SDK
- **Material You** — динамические цвета системы
- **SAF** (Storage Access Framework) — безопасный доступ к файлам
- **AlarmManager** — надёжное периодическое обновление виджета
- **Coroutines** — асинхронные операции

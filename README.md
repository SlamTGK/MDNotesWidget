**MD Notes Widget** — Android-приложение, которое переносит ваш «Второй Мозг» прямо на рабочий стол. Случайным образом выбирает и отображает Markdown (`.md`) файлы из любой папки на устройстве. Создано для интеграции с локальными базами **Obsidian** и другими Markdown-редакторами.

## Что нового в v3.0

### 🏷️ Фильтрация по тегам — полностью переработана
- **Тег-индекс** — при сканировании строится индекс `тег → файлы`, фильтрация мгновенная (без повторного чтения файлов)
- **ChipGroup для тегов** — вместо текстового поля теперь удобные чипы с кнопкой удаления и подсказками из реальных тегов
- **Корректный бейдж** — показывает "Найдено X из Y файлов (#тег)" вместо общего количества
- **Статус фильтра на виджете** — виджет показывает чип `🏷️ #тег` когда фильтр активен
- **Поддержка inline #hashtags** — теперь находит теги не только в YAML frontmatter, но и в теле заметки

### ✏️ Встроенный Markdown-редактор
- **FAB-кнопка** — переключает между просмотром и редактированием
- **Monospace-шрифт** для удобного редактирования кода
- **Автосохранение** — изменения записываются обратно в файл через SAF
- **Мгновенный рендеринг** — после сохранения заметка перерисовывается

### 📅 «В этот день» (On This Day)
- Новая вкладка в нижней навигации
- Показывает заметки, созданные/изменённые в этот день месяц/год назад
- Ностальгическая функция для обзора старых записей

### 🔍 Улучшенный поиск
- **Debounced-поиск** (300ms) — не перегружает устройство при быстром наборе
- **Подсветка совпадений** — найденные фрагменты выделены жёлтым в заголовке и превью
- **Единый адаптер** — поиск, история и избранное используют один NoteListAdapter

### 👆 Свайп-действия
- **← Свайп влево** в истории/избранном → удалить из списка (красный фон)
- **→ Свайп вправо** → открыть в Obsidian (фиолетовый фон)

### ⚡ Производительность
- **SAF ContentResolver.query()** — сканирование файлов в 10-50x быстрее (вместо DocumentFile.listFiles())
- **LRU-кеш** для отрендеренных SpannableStringBuilder — пролистывание без задержек
- **Bitmap downsampling** — изображения загружаются с inSampleSize, без OOM
- **Исправлен requestLayout()** — spannable-рендеринг обёрнут в `post {}`, нет мерцания
- **Thread-safe** SimpleDateFormat через ThreadLocal

### 🏗️ Архитектура
- **WidgetThemeHelper** — единая логика темизации для обоих виджетов
- **TagIndexManager** — менеджер тег-индекса с персистентностью в JSON
- **NoteListAdapter** — универсальный адаптер, заменивший 3 дублирующихся
- **FileOpener** — централизованное построение Obsidian URI
- **Unified error logging** — `Log.e` вместо `e.printStackTrace()`

## Ключевые возможности

### Виджеты

| Тип | Описание |
|-----|----------|
| **Случайная заметка** | Периодически показывает новую заметку, адаптирует количество строк под размер виджета |
| **Виджет-список** | Показывает заметку целиком с возможностью прокрутки |

- **Закрепление** — кнопка `📌` фиксирует текущую заметку
- **Создание** — кнопка `+` открывает диалог создания новой заметки
- **Статус тега** — чип с активными тегами прямо на виджете
- **Темы** — Фиолетовая, Тёмная, Прозрачная или свои цвета
- **Размер шрифта** — Мелкий / Средний / Крупный
- **Автоадаптация** — виджет подстраивает количество строк под свой размер

### Встроенный просмотрщик

- **Markdown-рендеринг** — заголовки с размером, **жирный**, _курсив_, ~~зачёркнутый~~, ☑ чекбоксы, • списки
- **Markdown-редактор** — редактирование заметок с monospace-шрифтом и автосохранением
- **Превью изображений** — горизонтальная прокрутка с изображениями из заметки (с downsampling)
- **Свайп** между случайными заметками
- **Поиск** — живой debounced-поиск с подсветкой совпадений
- **Избранное** — добавить/убрать одним нажатием + свайп-удаление
- **История** — последние 50 заметок + свайп-действия
- **«В этот день»** — заметки из этого дня в прошлые годы
- **Создание** — создать новую `.md` заметку
- **Меню ⋮** — поделиться, открыть в Obsidian, открыть во внешнем приложении, удалить

### Фильтрация

- **Теги (ChipGroup)** — удобный ввод через чипы с автоподсказками
- **Тег-индекс** — мгновенная фильтрация без чтения файлов
- **Frontmatter + inline** — поддержка YAML `tags:` и inline `#hashtags`
- **Логика AND/OR** — требуется один любой тег или все теги одновременно
- **Чёрный список папок** — исключить папки из выборки
- **Антиповтор** — последние 20 показанных заметок исключаются из рандома

### Настройки

- Интервал обновления: выкл / 15 мин / 30 мин / 1–24 часа
- Тихие часы — виджет не обновляется в заданный период
- Выбор действия при клике: встроенный просмотрщик / Obsidian / системный выбор

## Установка

1. Скачайте APK из [Releases](../../releases/latest)
2. Разрешите установку из неизвестных источников
3. Установите APK
4. Долгим нажатием на рабочий стол добавьте виджет **MD Notes Widget**
5. В настройках выберите папку с `.md` файлами

## Фильтр по тегам

Приложение поддерживает теги из YAML-заголовка Obsidian:

```yaml
---
tags:
  - избранное
  - daily
---
```

или

```yaml
---
tags: [избранное, daily]
---
```

А также inline-хештеги в теле заметки:
```markdown
Эта заметка про #рецепты и #кулинарию
```

Теги добавляются через ChipGroup в настройках — начните вводить и нажмите `+`.

## Требования

- Android 12+ (API 31)
- Разрешение на доступ к папке (выбирается через системный диалог)

## Технологии

- **Kotlin** + Android SDK
- **Material You** / Monet — динамические цвета системы + Material ChipGroup
- **SAF** (Storage Access Framework) — безопасный доступ к файлам (с оптимизированным ContentResolver.query)
- **AlarmManager** — надёжное периодическое обновление виджета
- **Coroutines** — асинхронные операции без блокировки UI

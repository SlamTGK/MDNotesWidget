**MD Notes Widget** — Android-приложение, которое переносит ваш «Второй Мозг» прямо на рабочий стол. Случайным образом выбирает и отображает Markdown (`.md`) файлы из любой папки на устройстве. Создано для интеграции с локальными базами **Obsidian** и другими Markdown-редакторами.

## Что нового в v2.5

- **🔍 Поиск** — встроенный поиск по названию и содержимому заметок прямо в просмотрщике
- **★ Избранное** — добавляйте заметки в избранное через меню ⋮, просматривайте в отдельном списке
- **➕ Создание заметок** — новая заметка из просмотрщика или по кнопке `+` на виджете, сохраняется в выбранной папке
- **⋮ Меню действий** — три точки в просмотрщике: удалить, поделиться, открыть в Obsidian/внешнем приложении
- **🗑️ Удаление с подтверждением** — безопасное удаление файла с диска
- **📤 Шаринг** — отправить текст заметки в любое приложение
- **🔁 Антиповтор** — рандомные заметки не повторяются подряд (история последних 20)
- **🏷️ Исправлен фильтр тегов** — поддержка YAML frontmatter (`tags: [tag1, tag2]`) и `tags:\n  - tag`)
- **📷 Исправлены изображения** — умная загрузка с тремя стратегиями поиска файла
- **📋 Исправлен Markdown** — корректное отображение заголовков, жирного, курсива, зачёркнутого
- **📜 Виджет-список** — исправлено наложение текста (StackView → ListView)

## Ключевые возможности

### Виджеты

| Тип | Описание |
|-----|----------|
| **Случайная заметка** | Периодически показывает новую заметку, адаптирует количество строк под размер виджета |
| **Виджет-список** | Прокручиваемый список карточек заметок |

- **Закрепление** — кнопка `📌` фиксирует текущую заметку
- **Создание** — кнопка `+` открывает диалог создания новой заметки
- **Темы** — Фиолетовая, Тёмная, Прозрачная или свои цвета
- **Размер шрифта** — Мелкий / Средний / Крупный
- **Автоадаптация** — виджет подстраивает количество строк под свой размер

### Встроенный просмотрщик

- **Markdown-рендеринг** — заголовки с размером, **жирный**, _курсив_, ~~зачёркнутый~~, ☑ чекбоксы, • списки
- **Превью изображений** — горизонтальная прокрутка с изображениями из заметки
- **Свайп** между случайными заметками
- **Поиск** — живой поиск по всем заметкам
- **Избранное** — добавить/убрать одним нажатием
- **История** — последние 50 просмотренных заметок
- **Создание** — создать новую `.md` заметку
- **Меню ⋮** — поделиться, открыть в Obsidian, открыть во внешнем приложении, удалить

### Фильтрация

- **Теги** — фильтрация по тегам из YAML frontmatter: `tags: [избранное]` или `tags:\n  - избранное`
- **Логика AND/OR** — требуется один любой тег или все теги одновременно
- **Чёрный список папок** — исключить папки из выборки (через `;`)
- **Антиповтор** — последние 20 показанных заметок исключаются из рандома

### Настройки

- Интервал обновления: выкл / 30 мин / 1–24 часа
- Тихие часы — виджет не обновляется в заданный период
- Выбор действия при клике: встроенный просмотрщик / Obsidian / системный выбор

## Установка

1. Скачайте APK из [Releases](../../releases/latest)
2. Разрешите установку из неизвестных источников
3. Установите APK
4. Долгим нажатием на рабочий стол добавьте виджет **MD Notes Widget**
5. В настройках выберите папку с `.md` файлами

## Фильтр по тегам

Приложение поддерживает теги из YAML-заголовка Obsidian:

```yaml
---
tags:
  - избранное
  - daily
---
```

или

```yaml
---
tags: [избранное, daily]
---
```

В поле фильтра вводите теги **без решётки `#`**, через `;`:
```
избранное; daily
```

## Требования

- Android 12+ (API 31)
- Разрешение на доступ к папке (выбирается через системный диалог)

## Технологии

- **Kotlin** + Android SDK
- **Material You** / Monet — динамические цвета системы
- **SAF** (Storage Access Framework) — безопасный доступ к файлам
- **AlarmManager** — надёжное периодическое обновление виджета
